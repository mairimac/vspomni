# Vspomni (Вспомни)
An adaptable language learning engine built with React and Tailwind CSS.

## Features
- **Modular Design**: Easily swap JSON data for different languages.
- **Hard Mode**: Study only flagged cards.
- **Virtual Russian Keyboard**: Built-in support for Cyrillic input.
- **Persistence**: Progress and streaks saved via LocalStorage.

## Tech Stack
- React (Vite)
- Tailwind CSS
- Lucide React (optional)
